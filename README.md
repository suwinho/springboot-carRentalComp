# Car Rental System

A simple and functional web application for managing a car rental business, built with the Spring Boot framework.

## üìã Project Description

This application enables comprehensive management of the vehicle rental process. It allows you to maintain a database of cars, manage customer information, and handle reservations/rentals. The project follows the MVC pattern, where the backend is based on Java (Spring Boot) and the frontend is rendered using a template engine (Thymeleaf/HTML).

## üöÄ Features

* **Fleet Management:**
    * Add new cars to the system.
    * Edit vehicle details.
    * Remove vehicles from the database.
    * View the list of available cars.
* **Rental Operations:**
    * Create new reservations.
    * Track rental status.
    * Assign cars to customers.
* **Customer Management:**
    * Register and edit customer details.
